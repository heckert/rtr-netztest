# rtr-netztest

Exploring gradient boosting algorithms on internet speedtest data.
> __NOTE:__ _Currently still work in progress_

## Setup environment
    make environment
will create a conda environment with the packages listed in `environment.yml`.

## Run tests
    make tests
will run the tests.
